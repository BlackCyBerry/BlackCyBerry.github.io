SessionDescriptor

A descriptor for an API session that can be used to initialize and connect to an API instance. Think of it as a file handle describing an open API session.

Even though this object does not contain any transient details and would be safe to persist, this is not a preferred pattern; see IConsoleConfig and ConfigurationManager for means of persisting credentials.
