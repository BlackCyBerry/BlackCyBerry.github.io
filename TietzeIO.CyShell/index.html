    What's this?¶

    TietzeIO.CyShell is a Powershell module for interacting with the Cylance REST API written in C#.

    Core scenarios¶

    Core usage scenarios for this module are:

    • Automation of common administrative tasks, e.g. mass operations or migrations
    • To serve as a sample application for the TietzeIO.CyAPI set of libraries

    Why use it?¶

    What sets it apart from other modules (e.g. CyCLI1) the most is:

    • Performance - multiple threads, connection pools etc.
    • Reliability - resilient access to backend APIs
    • Type safety - no more dealing with dynamic objects
    • Ease of use - as a result

    Please update!


    CyShell is the successor module to CyCLI, and you are encouraged to move your code to CyShell. If you are interested in CyCLI, you can find it here1

    1. CyCLI and CyShell have the same origin and author - CyShell is meant as CyCLI's successor and to be better in every aspect. ↩↩
